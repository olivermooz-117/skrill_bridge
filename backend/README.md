RESTful API for the SkillBridge freelance marketplace platform.

##  Features

- User Authentication (JWT)
- Gig Management (CRUD)
- Order Management
- Reviews & Ratings
- Password Reset Flow
- Role-Based Access Control

##  Tech Stack

- Flask 2.3.3
- Flask-SQLAlchemy
- Flask-JWT-Extended
- SQLite / PostgreSQL
- Flask-CORS
- Flask-Migrate

## Prerequisites

- Python 3.8+
- pip
